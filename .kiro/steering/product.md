# Product Overview

RivaAI is a telephony-based cognitive voice interface that provides decision support through natural phone conversations. Designed for users with basic feature phones in rural environments requiring no digital literacy.

## Core Capabilities

- Full-duplex streaming voice interface with barge-in support
- Multi-language support (Hindi, Marathi, Telugu, Tamil, Bengali)
- RAG-based decision intelligence using domain-specific knowledge bases
- Safety-first design with circuit breaker mechanisms
- Privacy-preserving with automatic PII masking and 24-hour data retention

## Target Users

Users with basic feature phones in rural environments who need decision support but have limited or no digital literacy.

## Key Design Principles

1. **Accessibility First**: Works on basic PSTN phones, no smartphone required
2. **Safety First**: Circuit breakers prevent harmful outputs
3. **Privacy Preserving**: Automatic PII masking, 24-hour data retention
4. **Low Latency**: Optimized for real-time conversation with <500ms response times
5. **Multi-language**: Native support for major Indian languages
